# 🚢 SBES Professional Data Processor

An automated, web-based tool for cleaning and processing **Single Beam Echo Sounder (SBES)** raw data files. Designed specifically for hydrographic surveyors using HYPACK or similar systems.

## ✨ Key Features
- **Automated Despiking:** Statistical Hampel filter to remove acoustic noise.
- **Vertical Corrections:** Apply Tide and Transducer Draft offsets.
- **Spatial QC:** Interactive 2D map color-coded by depth to verify coverage.
- **Batch Export:** Process individual files or the entire project into clean `.xyz` files.
- **Modern UI:** Clean, tabbed interface for professional user experience.
